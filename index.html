Projects

Game Galaxy

Game Galaxy was my front-end capstone project for Nashville Software School. I wanted to build a social blogging site centered on sharing information and opinions on video games. The tools I used included AngularJS, Lodash, JQuery, Sass, Bootstrap, and more. Check out the source code and readme on Github.


Asciimals

A very light tool created so I could display ASCII animal art in my Terminal. Built using NodeJS.


Popcorn Brothers Podcast

A movie review and discussion podcast I cohost with my older brother. An good example of how my skills as a web developer have grown.
