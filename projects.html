Festivus Translator

A translator that uses modularized Javascript to translate a classic Festivus quote from English to one of four other languages. All styling was written with Sass and the page includes original CSS Animation, as well as Bootstrap and the HTML5 Speech API, spoken in a random voice from the preset list. This was the first, complete project I had worked on using Grunt. Image editing was performed using Adobe Photoshop. Git and Github were used as version control system for feedback and updates.
